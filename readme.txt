Ada Tamil Writer provides easy Tamil typing options in WordPress, with layouts like Thaminglish, Tamil99, and Typewriter.

== Description ==

Ada Tamil Writer is a powerful plugin that allows you to type in Tamil directly from your WordPress admin area. The plugin adds a custom admin bar menu where you can easily switch between different Tamil keyboard layouts, such as Thaminglish, Tamil99, and Typewriter.

Key features:
* Quick switch between Tamil keyboard layouts in the admin bar.
* Multiple Tamil keyboard layouts supported - including Thaminglish, Tamil99, and Typewriter.
* As admin, you can set the default keyboard layout for the site and visitors can configure their own preference.
* You can type Tamil directly within many fields of WordPress without installing any Tamil fonts
* Let your site visitors type Tamil - such as in Comments - without needing them to install anything
* Customizable settings for logged-in and non-logged-in users.
* Supports English typing as well.
* Requires the Classic Editor plugin for optimal functionality.
* Clean and modern design that integrates seamlessly with the WordPress admin interface.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/ada-tamil-writer` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to the "Ada Tamil Writer" menu in the WordPress admin sidebar to configure the plugin settings.
4. Ensure the Classic Editor plugin is installed and activated for the best experience.

== Frequently Asked Questions ==

= What keyboard layouts are supported? =

Ada Tamil Writer supports multiple Tamil keyboard layouts including Thaminglish, Tamil99, and Typewriter

= How do I switch between keyboard layouts? =

After installing and activating the plugin, you'll see an "Ada Tamil Writer" option in your admin bar. From there, you can select your preferred keyboard layout.

= Can I use this plugin with the Gutenberg editor? =

The plugin is primarily designed for use with the Classic Editor. Some features may not work as expected with the Gutenberg editor.

= Do I need the Classic Editor plugin? =

Yes, the Classic Editor plugin is recommended for optimal functionality. The plugin will prompt you to install and activate Classic Editor if it's not already installed.

= What happens if the Classic Editor plugin is not activated? =

If the Classic Editor plugin is installed but not activated, you will be prompted to activate it. If it's not installed, the plugin will recommend installation for the best experience.
